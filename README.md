# 🎓 EduSpark – Learning Management System

EduSpark is a full-stack Learning Management System (LMS) that provides an interactive online learning experience. The platform supports two user roles—**Admin** and **User**—allowing administrators to manage educational content while students can access courses and lecture videos through a modern web interface.

---

## 📖 Overview

EduSpark is an educational platform developed to simplify online learning and course management. It enables administrators to upload courses, organize lecture videos, and manage educational content, while users can browse, access, and learn from available courses.

The application follows a modern client-server architecture using **React** for the frontend and **Node.js** for the backend, ensuring a responsive, scalable, and user-friendly learning environment.

## ⚙️ System Architecture

The application consists of two main modules:

### 👨‍💼 Admin Module

- Secure Login
- Upload Courses
- Upload Lecture Videos
- Manage Educational Content
- Update Course Information

---

## 🔄 Workflow

1. Admin logs into the dashboard.
2. Admin uploads courses and lecture videos.
3. Course information is stored in the database.
4. Users register and log into the platform.
5. Users browse available courses.
6. Users watch lectures and access learning materials.
